## Run Locally

Clone the project

```bash
  git clone https://github.com/lobnor2/Tibet-Film.git
```

Go to the project directory

```bash
  cd Tibet-Film
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Features

- Light/dark mode toggle
- Favorite / Watchlist
- Ai voice control
- Cross platform
